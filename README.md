# bryanseah234.github.io

A personal GitHub Pages website featuring interactive web experiments, mini-games, and project showcases.

## Description

This repository hosts a collection of interactive web projects built with vanilla HTML, CSS, and JavaScript. It includes a Coca-Cola-inspired mobile drinking game, a Blackjack card game, a fan community page with voting and chat features, and a landing page for sgRails - a conceptual train commuting app for Singapore.

## Features

- **Coca-Cola Drinkable Ad Game**: Mobile-optimized interactive game where users tap or shake to drink from a virtual glass with realistic bubbles and ice cube animations
- **Blackjack Game**: Classic card game implementation with start game and draw card functionality
- **Justin Bieber Fan Page**: Interactive fan community page featuring song voting, real-time chat simulation, and image gallery with modal view
- **sgRails Landing Page**: Modern responsive landing page showcasing a conceptual smart train commuting app with AR navigation features
- **Random Profile Picture Generator**: Fun button that displays random GitHub user avatars

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations, Custom Properties)
- Vanilla JavaScript (ES6+)
- Tailwind CSS (for sgRails landing page)
- Device Motion API (for shake detection in Coca-Cola game)
- Local Storage API (for game high scores)

## Installation

```bash
# Clone the repository
git clone https://github.com/bryanseah234/bryanseah234.github.io.git

# Navigate to project directory
cd bryanseah234.github.io

# No build steps required - open any HTML file directly in a browser
```
