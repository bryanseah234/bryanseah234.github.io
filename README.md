# bryanseah234.github.io

A collection of interactive web experiments and mini-projects hosted on GitHub Pages.

## Description

This repository showcases a variety of fun, interactive web applications built with vanilla HTML, CSS, and JavaScript. Each project demonstrates different web development techniques including mobile-first design, game development, responsive layouts, and interactive UI components. The projects range from simple games to landing pages and fan sites.

## Features

- **Personal Landing Page** - A simple profile page with interactive buttons and random GitHub avatar generator
- **Coca-Cola Drinkable Ad Game** - A mobile-first interactive game where users tap or shake their device to "drink" a virtual cola (inspired by Coca-Cola's drinkable ad campaign)
- **Justin Bieber Believers Fan Page** - An interactive fan site with song voting, live chat simulation, and image gallery with modal viewer
- **Blackjack Card Game** - A classic casino-style card game implementation
- **sgRails Landing Page** - A concept landing page for a Singapore train commuting app with modern UI/UX design, featuring AR navigation and real-time alerts

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations, Transitions)
- Vanilla JavaScript (ES6+)
- Tailwind CSS (for sgRails page)
- Web APIs (Device Motion, LocalStorage, Clipboard)

## Installation

```bash
# Clone the repository
git clone https://github.com/bryanseah234/bryanseah234.github.io.git

# Navigate to project directory
cd bryanseah234.github.io

# Open any HTML file in your browser
# No build step or dependencies required!
```

## Usage

Simply open any of the HTML files in a web browser:

- `index.html` - Main personal page
- `cola.html` - Coca-Cola drinkable game (best experienced on mobile)
- `fanpage.html` - Justin Bieber fan page
- `gamble.html` - Blackjack card game
- `rails.html` - sgRails landing page
